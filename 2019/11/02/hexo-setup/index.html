Hexo setup

This post is how I deploy hexo into github.io.

Setup hexo

Now open http://localhost:4000 to see the effect.

Install icarus theme

deploy to github

create a repo call chihungtzeng.github.io in github web site.

Install hexo-deployer-git to enable hexo deploy.

go to github web site and register ssh keys: settings->SSH and PGP keys

From now on, we can simply run hexo deploy to deploy our repo to github.io.

Other icarus configs like qrcode are left for you to config.
